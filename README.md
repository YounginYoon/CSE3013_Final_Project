`게임 방법`

<br/>

**1. Map 선택**
   
  * File → Open으로 .maz 파일을 선택합니다. (5_5.maz, 10_10.maz, 12_8.maz, maze0.maz 4 개 중 택 1)
<br/>
 
**2. 키보드 조작**

  * s : 게임 시작
  
  * 캐릭터 이동 : 방향키(↑, ↓, →, ←)
  
  * r : 힌트 제공 (도착 지점까지 최단 경로, 3초동안 제공)
  
  * e : 게임 재시작을 위해 고양이 시작 위치로 이동
  
  * q : 게임 종료
<br/>

**3. 제한 시간**

  * 5_5.maz: 15초
  
  * 그 외 파일: 20초
